# Task 132: next/font 타이포그래피 시스템
상태: 미구현 | Phase 3 | 예상 시간: 3h
의존성: 112 (결정론적 스캐폴드)

## 1. 태스크 정의
vibeDeploy가 생성하는 앱의 가독성과 시각적 위계를 위해 `next/font/google`을 활용한 타이포그래피 시스템을 구축합니다. 도메인별 10가지 폰트 페어링(Display + Body)을 정의하고, 이를 `layout.tsx`에 자동 주입하여 일관된 타입 스케일을 제공합니다.

## 2. 수용 기준 (Acceptance Criteria)
- [ ] AC-1: 10가지 폰트 페어링(Editorial, Modern SaaS, Elegant, Tech/Dev, Friendly, Business, Creative, Minimal, Classic, Bold) 정의
- [ ] AC-2: `blueprint.design_system.typography` 키워드 매칭을 통한 최적 페어링 선택 로직 구현
- [ ] AC-3: `layout.tsx`에 유효한 `next/font/google` import 및 변수 설정 생성
- [ ] AC-4: CSS 변수 `--font-display`, `--font-body`를 통한 폰트 패밀리 제어
- [ ] AC-5: 9단계 타입 스케일(display-xl ~ caption) 정의 및 `globals.css` 주입

## 3. 변경 대상 파일
- `agent/nodes/build.py`: `select_font_pairing`, `generate_layout_tsx` 함수 추가
- `web/src/app/layout.tsx`: 생성된 폰트 설정이 주입될 대상 파일

### 4.2 페어링 선택 및 레이아웃 생성
```python
def select_font_pairing(typography_hint: str) -> dict:
    hint = typography_hint.lower()
    for pairing in FONT_PAIRINGS:
        if any(kw in hint for kw in pairing["keywords"]):
            return pairing
    return FONT_PAIRINGS[1] # Default: Modern SaaS

def generate_layout_tsx(pairing: dict, meta: dict) -> str:
    display = pairing["display"]
    body = pairing["body"]
    
    return f"""
import {{ {display}, {body} }} from 'next/font/google';
import './globals.css';

const displayFont = {display}({{ 
  subsets: ['latin'], 
  variable: '--font-display',
  weight: ['700', '800'] 
}});

const bodyFont = {body}({{ 
  subsets: ['latin'], 
  variable: '--font-body',
  weight: ['400', '500'] 
}});

export const metadata = {{
  title: '{meta.get("title", "vibeDeploy App")}',
  description: '{meta.get("description", "Generated by vibeDeploy")}',
}};

export default function RootLayout({{ children }}: {{ children: React.ReactNode }}) {{
  return (
    <html lang="ko" className="{{`${{displayFont.variable}} ${{bodyFont.variable}}` font-sans text-foreground bg-background antialiased"}}>
      <body className="min-h-screen flex flex-col">
        {{children}}
      </body>
    </html>
  );
}}
"""
```
