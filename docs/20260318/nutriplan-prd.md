## 1. Product Overview

| Item | Detail |
|------|--------|
| **Name** | NutriPlan |
| **One-liner** | "Enter your weight and goal, AI tells you what to eat today" |
| **Core Problem** | Existing nutrition apps (MyFitnessPal, Noom) require daily manual food logging → 70% churn within 2 weeks |
| **Solution** | Weight + goal input → AI calculates macros → auto-generates daily meal plan → one-click grocery list |
| **Target User** | Fitness beginners (20-35) who started exercising but quit nutrition management due to complexity |
| **Positioning** | "Just tell me what to eat" — meal planning without calorie counting |

---

## 2. Key Insights from Video

1. **Simplification drives sustainability** — Weight-based formula (15-17 cal/lb) + macro ratio (1g protein/lb) instead of complex calorie counting
2. **Fixed templates eliminate decision fatigue** — 4-5 meals/day fixed pattern, only ingredients vary
3. **Grocery list creates daily utility** — Auto-generated list saves 45min/week → reason to open app daily

## 6. Data Model

```
User
  |-- id, email, password_hash
  |-- weight, goal (cut/maintain/bulk)
  |-- dietary_restrictions[]
  |-- created_at

MacroTarget
  |-- user_id (FK)
  |-- daily_calories, protein_g, carbs_g, fat_g
  |-- calculated_at

MealPlan
  |-- id, user_id (FK)
  |-- date, meals[] (JSON)
  |-- is_saved (boolean)
  |-- generated_at

GroceryList
  |-- plan_id (FK)
  |-- items[] (JSON: name, quantity, category)
  |-- generated_at
```

## 7. API Design

```
POST   /api/auth/register         — Register
POST   /api/auth/login             — Login
POST   /api/calculate              — Macro calculation (weight, goal → calories, macros)
POST   /api/generate-plan          — AI meal generation (macro_target_id → meals[])
GET    /api/plans                   — List my plans
GET    /api/plans/{id}             — Plan detail
PATCH  /api/plans/{id}/meals/{idx}  — Swap specific meal (AI regeneration)
POST   /api/plans/{id}/save        — Bookmark plan
GET    /api/grocery-list/{plan_id}  — Generate grocery list
```

## 8. AI Prompt (OpenAI)

```
You are a sports nutrition AI. Given the user's macro targets:
- Daily calories: {calories}kcal
- Protein: {protein}g, Carbs: {carbs}g, Fat: {fat}g
- Dietary restrictions: {restrictions}

Generate a simple daily meal plan with exactly {meal_count} meals.

Each meal must include:
- Name (simple, home-cookable)
- Ingredients with gram amounts
- Macro breakdown (protein_g, carbs_g, fat_g)

Rules:
- Use common grocery store ingredients only
- Keep prep time under 20 minutes per meal
- Total macros must be within +/-5% of targets
- Vary protein sources across meals

Return as JSON array.
```
